# ps1.git

ps1 is a dual polarity (+/- 15V) full wave rectified power supply. Schematic capture and PCB layout conducted in KiCad.

Some [requirements](../../wiki/Power-Supply-Requirements) are in the works over in the [wiki](../../wiki).
